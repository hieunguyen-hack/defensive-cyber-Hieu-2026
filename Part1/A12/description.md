1. VeraCrypt
You can encrypt an entire external drive, take it to a remote cabin, and access your files using a local key or a PIM (Personal Iterations Multiplier) for extra brute-force protection. it requires 0 wifi connection.

2.KeePassXC:

can be used on Windows, MacOS, Linux
Your passwords remain encrypted at all times and no data is stored on remote servers, so you stay in full control of your data.
Your passwords are stored in a local encrypted file (.kdbx)
Everything (encryption, access, storage) happens on your device

4. Autopsy:

It is designed to analyze hard drive images without ever mounting them to a live OS (which prevents accidental data alteration). You can take a physical snapshot of a suspicious laptop, move that file to your "clean" offline machine, and reconstruct everything: deleted files, web history, and even timeline views of when certain USB drives were plugged in.

5.GnuPG

GnuPG is a hybrid-encryption software program because it uses a combination of conventional symmetric-key cryptography for speed, and public-key cryptography for ease of secure key exchange, typically by using the recipient's public key to encrypt a session key which is used only once.
