
*Device Security & Physical Access
Screen Locks: Using strong passcodes, fingerprints, or face recognition for all mobile devices and computers.
Privacy Screens: Using anti-glare, matte, or privacy filters on laptops and phones to prevent "shoulder surfing"


*Offline Security Strategies
Limiting Data Storage: Only keeping essential information and destroying it when no longer required.
Role-Based Access: Restricting access to physical files and devices in an office environment.
