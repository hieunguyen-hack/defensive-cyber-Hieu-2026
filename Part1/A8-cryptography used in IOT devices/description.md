IoT cryptography uses the same principles and algorithms as modern network cryptography, but it is optimized with:

lighter algorithms - Examples: PRESENT/SIMON/SPECK/ASCON (NIST lightweight crypto winner)

smaller keys (ECC): a type of public key cryptography, using a public-private key pair

lightweight protocols (MQTT, CoAP, DTLS)

to work on low-power devices.

the most important role for cryptography in IOT is to secure the communications
